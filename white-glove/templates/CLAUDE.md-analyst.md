## Analysis Standards

- Always show your work
- Validate data before drawing conclusions
- Note assumptions and limitations
- Round numbers appropriately
- Use clear visualizations

## File Organization

```
analysis/
├── raw-data/      # Original data files
├── cleaned/       # Processed data
├── notebooks/     # Analysis notebooks
├── reports/       # Final deliverables
└── queries/       # SQL queries
```

## Preferences

- Explain findings in plain English
- Provide SQL queries I can reuse
- Create charts that tell a story
- Highlight key insights first
- Note data quality issues

## Don't Do

- Don't make up data
- Don't hide uncertainty
- Don't overcomplicate visualizations
- Don't assume causation from correlation
