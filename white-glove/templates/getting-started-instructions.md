# Getting Started with Claude Code
*Your pre-session setup - 15 minutes*

---

## What You'll Do

Claude Code will guide you through setting up your entire stack interactively. No need to figure things out alone - Claude walks you through each step.

---

## Step 1: Create Your Project Folder

Open Terminal and run:

```
mkdir ~/claude-workspace
cd ~/claude-workspace
```

---

## Step 2: Download the Setup File

Save the attached `CLAUDE.md` file to your project folder:

```
~/claude-workspace/CLAUDE.md
```

(You can drag and drop it into Finder, or use Terminal)

---

## Step 3: Start Claude Code

In Terminal, make sure you're in the folder:

```
cd ~/claude-workspace
claude
```

---

## Step 4: Follow Claude's Guide

Claude will automatically start the setup questionnaire:

1. **Basic check** - Verifies Node, GitHub CLI are installed
2. **Stack discovery** - Asks what services you use
3. **Credential setup** - Walks you through getting each API key
4. **Verification** - Tests each connection works
5. **Report** - Shows what's ready, what needs fixing

Just answer Claude's questions. It handles the rest.

---

## What You'll Need Handy

Login credentials for services you use:
- GitHub (github.com)
- Supabase (supabase.com) - if you use it
- n8n (your instance URL) - if you use it
- Any other services in your stack

Claude will tell you exactly where to find each API key.
